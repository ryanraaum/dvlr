dvlr 0.4.0

Files included:

dvlr.py:        core of the program
dvlr_gui.py:    graphical interface (using Tkinter)
setup.py:       python distutils script

Requires:

Python 2.2 or later
numarray (http://www.stsci.edu/resources/software_hardware/numarray)

Windows executable built using py2exe
(http://starship.python.net/crew/theller/py2exe/)

To build Windows executable run this command in the source directory:
'python setup.py py2exe -p numarray'

Packaged for distribution using Advanced Installer (www.advancedinstaller.com)
